面试官：你有什么想问的？

技术

  • 任意页面布局是否可以不运行App主逻辑进行直接运行到真机调试看效果吗？
  • 在主工程可以通过调整开关来调试已经发布为aar依赖的子工程的源码吗？
  • 主工程和多个子工程依赖相同库的版本号不同步解决了吗？
  • 编译完整的项目的时间会超过10分钟吗？
  • 每次发布新版本，会有工具自动检测编译时间的增加的原因和Apk体积增大的原因吗？
  • App发生了崩溃，会立刻自动建JIRA指派给对应代码的研发负责人吗？
  • 日志库实际会打印大量无效的日志吗？
  • 产线包里输出的每一行日志是否都有自定义的错误码？
  • 日志内容对于测试、客服等非开发人员是否可读？
  • 产品经理会对功能描述给出详细的测试用例吗？
  • 设计图中UI控件的样式给出的是主题中的某个特定样式名吗？还是零散的属性值？
  • 布局都在用ConstraintLayout了吗？
  • Kotlin协程在用吗？
  • 单元测试覆盖率有强制要求吗？
  • git要求原子化提交吗？
  • git message要求写出每一个改动的原因吗？
  • git push代码后，CI会编译项目校验吗？
  • 每个人可以看到所有代码库的代码吗？
  • 一线的Leader有做过独立开发吗？有下载量大的作品吗？

办公环境

  • 工位之间有隔断吗？
  • 业务相关的上下游部门员工是否坐在同一个区域？
  • 工位附近的白板多吗？
  • 坐在附近的员工平时面对面交流的多吗？
  • 每一层办公区是一个大开间吗？站起身来是否可以看到这一层的任意一个人在不在位置上？
  • 每一层办公区只有一到两条长走廊吗？
  • 每一层只有一个茶水区和卫生间吗？
  • 领导有单独的办公室吗？
